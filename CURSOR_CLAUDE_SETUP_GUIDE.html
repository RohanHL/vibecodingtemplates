Part 1: Download and Install Cursor

Cursor is a code editor built on top of VS Code with AI features built in. You'll use its integrated terminal to run Claude Code.

System Requirements

  • Windows: Windows 10 or 11
  • macOS: macOS 10.15 (Catalina) or newer
  • Linux: Ubuntu 20.04+ or Debian-based distributions
  • RAM: 4GB minimum (8GB+ recommended)
  • Disk Space: At least 500MB free

Download Cursor

Step 1: Visit https://cursor.com/download

The website will auto-detect your operating system and provide the correct download.

Install Cursor

Windows Installation

  1. Locate the downloaded .exe file (usually in Downloads folder)
  2. Right-click and select "Run as Administrator"
  3. If Windows SmartScreen appears:
    • Click "More info"
    • Click "Run anyway"
  4. Follow the installation wizard (accept defaults)
  5. Wait 1-2 minutes for installation
  6. Launch Cursor when complete

Alternative via Chocolatey:

choco install cursor -y

macOS Installation

  1. Locate the downloaded .dmg file
  2. Double-click to open it
  3. Drag the Cursor icon to the Applications folder
  4. Open Applications and double-click Cursor
  5. If you see a security warning:
    • Go to System Preferences > Security & Privacy
    • Click "Open Anyway"
  6. Click "Open" to confirm

Linux Installation (Ubuntu/Debian)

Option A: Using .deb package (recommended)

cd ~/Downloads
sudo dpkg -i cursor-*.deb
sudo apt-get install -f  # Fix dependencies

Option B: Using AppImage (portable)

cd ~/Downloads
chmod +x Cursor-*.AppImage
./Cursor-*.AppImage

Initial Cursor Setup

  1. Launch Cursor
  2. Sign up or sign in (use email, Google, or GitHub)
  3. Choose your theme and settings (optional)
  4. Import VS Code settings if you have them (optional)
‚úì Cursor is now installed! You'll see an interface similar to VS Code.

Authenticate Claude Code

  1. In Cursor's terminal, run: 
    claude /login
  2. Choose your authentication method (Claude App is most common)
  3. A browser window opens automatically
  4. Log in to Claude.ai and click "Authorize"
  5. Return to terminal - you'll see "‚úì Authentication successful"

cd test-claude

You should see the Claude Code prompt. Try asking:

Type /exit to quit.

Part 3: Set Up Perplexity MCP

The Perplexity MCP allows Claude Code to search the web in real-time.

Get Perplexity API Key

  1. Go to https://www.perplexity.ai
  2. Sign in or create an account
  3. Navigate to Settings > API
  4. Click "Generate API Key" or copy existing key
  5. Save it somewhere safe

Install UV Package Manager

PowerShell (in Cursor's terminal):

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

With Homebrew:

brew install uv

Without Homebrew:

curl -LsSf https://astral.sh/uv/install.sh | sh
curl -LsSf https://astral.sh/uv/install.sh | sh

Verify UV installation:

uv --version

Install Perplexity MCP

Run this command (replace YOUR_API_KEY_HERE with your actual key):

claude mcp add --transport stdio perplexity --scope user --env PERPLEXITY_API_KEY=YOUR_API_KEY_HERE -- uvx perplexity-mcp

Understanding this command:

  • --transport stdio = Local process communication
  • perplexity = Server name
  • --scope user = Available across all projects
  • --env = Sets your API key securely

Verify installation:

claude mcp list

You should see perplexity listed.

Test Perplexity Integration

cd ~/Desktop/test-claude
claude

> Search the web for the latest updates on Claude AI from the past week

Claude will ask for permission. Type A (Allow once) and press Enter.

‚úì Perplexity MCP is now working! Claude can now search the web for you.

Part 4: Understanding Configuration Scopes

MCP Scopes Explained

You installed Perplexity at user scope. Here's what that means:

Scope Location When to Use
User
(What you did)		 ~/.config/claude-code/mcp.json Tools you use across all projects (like Perplexity)
Project .mcp.json in project folder Team-shared tools (gets committed to git)
Local Project-specific, private Testing or temporary needs
‚ö†Ô∏è Security Note: Never commit API keys to git! Use user scope for personal keys or environment variables for team projects.

Configuration Files

Your Perplexity configuration is stored here:

  • macOS/Linux: ~/.config/claude-code/mcp.json
  • Windows: %APPDATA%\claude-code\mcp.json

To view it:

cat ~/.config/claude-code/mcp.json

Troubleshooting Common Issues

Claude Code Not Found

Problem: command not found: claude

Solution: Reload your shell

source ~/.bashrc  # Linux
source ~/.zshrc   # macOS
# Or close and reopen Cursor's terminal

UV Not Found

Problem: command not found: uv

Solution: Re-install and reload

curl -LsSf https://astral.sh/uv/install.sh | sh
source ~/.bashrc  # or ~/.zshrc

Perplexity MCP Not Working

Problem: "Tool 'perplexity_search' not available"

Solution: Check and re-add if needed

claude mcp list

# If not listed, add again:
claude mcp add --transport stdio perplexity --scope user --env PERPLEXITY_API_KEY=YOUR_KEY -- uvx perplexity-mcp

API Key Invalid

Problem: "Authentication failed"

Solution: Update with new key

claude mcp remove perplexity
claude mcp add --transport stdio perplexity --scope user --env PERPLEXITY_API_KEY=NEW_KEY -- uvx perplexity-mcp

Permission Denied (macOS)

Solution:

# For Cursor
xattr -cr /Applications/Cursor.app

# For Claude Code
chmod +x $(which claude)

üìñ Glossary: What Do These Words Mean?

Don't know what these technical terms mean? No problem! Here's a simple explanation of everything:

üñ•Ô∏è Basic Computer Terms

Terminal / Command Line
A black or white box where you type text commands instead of clicking buttons. Think of it like texting with your computer. It might look scary, but we'll tell you exactly what to type!

Install / Installation
Putting new software (programs) on your computer. Like downloading an app on your phone, but for your computer.

Download
Copying a file from the internet to your computer. Usually goes to your "Downloads" folder.

Code Editor
A program for writing code (computer instructions). Like Microsoft Word, but designed for programming instead of writing documents.

Directory / Folder
Same thing! A place on your computer where files are stored. "Directory" is the technical word for "folder".

ü§ñ AI & Coding Terms

Cursor
The code editor (program) you'll install. It's like VS Code (another popular editor) but with AI features built in.

Claude Code
An AI assistant made by Anthropic that helps you write code. You talk to it in the terminal, and it helps you build things.

API Key
A special password that lets one program talk to another. Think of it like a secret handshake between apps. You'll need one for Perplexity.

MCP (Model Context Protocol)
A way for AI assistants to connect to other tools and services. Don't worry about the technical details - just know it lets Claude Code do more things (like web search).

Perplexity
A web search service. Adding it lets Claude Code search the internet for current information.

‚öôÔ∏è Technical Terms You'll See

Package Manager
A tool that installs other tools automatically. Like an app store, but for developer tools. We use one called "UV".

Command
Text you type in the terminal to make the computer do something. We'll always show you exactly what to type.

Authentication / Login
Proving who you are. Like logging into any website, but for Claude Code.

Scope (User, Project, Local)
Where settings are saved:

  • User: Works everywhere on your computer (recommended)
  • Project: Only works in one specific project folder
  • Local: Private settings just for you in one project

Environment Variable
A setting the computer remembers. Like saving your preferences. Don't worry - we'll set these up for you.

Repository / Repo
A folder where code is stored (often on GitHub). Think of it as a shared folder for code projects.

GitHub
A website where people share code. Like Dropbox or Google Drive, but specifically for code projects.

üíª Operating System Specific

WSL (Windows Subsystem for Linux)
A way to run Linux commands on Windows. Only needed for Windows users. It's optional but helpful.

Git Bash
A program for Windows that lets you use terminal commands. An alternative to WSL.

Homebrew
A package manager for Mac. Makes installing developer tools easier. It's optional - we show other ways too.

Admin / Administrator
Full permissions to make changes to your computer. You'll need this password to install software.

PATH
A list of places your computer looks for programs. When we "add to PATH," we're making a program available everywhere.
