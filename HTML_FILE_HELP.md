# How to Open HTML Files - Troubleshooting Guide

## üòü Problem: HTML File Shows Code Instead of a Webpage

If you downloaded an HTML file and when you open it, you see a bunch of code like this:
```html
<!DOCTYPE html>
<html lang="en">
<head>
...
```

**This means it opened in a text editor instead of a web browser.**

---

## ‚úÖ Solution: Open with Your Web Browser

### Method 1: Right-Click and Choose Browser (Easiest)

1. **Find the HTML file** (it's in your Downloads folder)
2. **Right-click** on the file (or Control+Click)
3. **Choose "Open With"**
4. **Select your web browser:**
   - Safari (comes with Mac)
   - Chrome
   - Firefox
   - Edge
5. **Click "Open"**

The file should now open as a nice-looking webpage!

---

### Method 2: Drag and Drop into Browser

1. **Open your web browser** (Safari, Chrome, etc.)
2. **Open a new tab** (Command + T)
3. **Find the HTML file** in your Downloads folder
4. **Drag the file** into the browser window
5. **Drop it**

The file will open as a webpage!

---

### Method 3: Change Default Program (Future Files)

**So HTML files always open in your browser:**

1. **Find the HTML file**
2. **Right-click** (or Control+Click)
3. **Choose "Get Info"**
4. **Find "Open with:"** section
5. **Click the dropdown**
6. **Select your browser** (Safari, Chrome, etc.)
7. **Click "Change All..."**
8. **Click "Continue"**

Now all HTML files will open in your browser automatically!

## üÜò Still Not Working?

### Problem: File downloaded as .txt instead of .html

**Check the file name:**
- It should end in `.html`
- If it ends in `.txt`, rename it

**To rename:**
1. Right-click the file
2. Choose "Rename"
3. Change the extension from `.txt` to `.html`
4. Press Enter
5. Click "Use .html" when warned

---

### Problem: Browser says "File Not Found"

**This means:**
- The file was moved or deleted
- Download it again
- Make sure you know where it's saved (check Downloads folder)

---

### Problem: Page looks broken (no colors, weird layout)

**This might happen if:**
- The file wasn't downloaded completely
- Download it again
- Try a different browser
